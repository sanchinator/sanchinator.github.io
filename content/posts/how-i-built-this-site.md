## How I add a new post

Adding content is straightforward:

1. Create a new `.md` file in the `content/posts/` folder
2. Write the reflection in Markdown
3. Commit and push to GitHub
4. GitHub Actions builds and deploys automatically

The whole process takes less than a minute once the content is written.
